# Object-Detection-Classification
# 包装OK/NG智能检测系统

一个基于深度学习的包装质量自动检测系统，能够快速识别包装的合格(OK)与不合格(NG)状态。

## 项目简介

本项目旨在利用计算机视觉技术实现包装质量的自动化检测，通过深度学习模型对包装图像进行分析，判断其是否符合质量标准。系统具有以下特点：

- 基于ResNet18深度学习模型，检测准确率高
- 提供直观的图形用户界面，操作简单易用
- 支持图像数据自动分类与数据集划分
- 实时显示检测结果及置信度，方便人工复核
- 生成可视化概率分布图表，直观展示模型判断依据

## 功能说明

1. **数据处理**：
   - 自动筛选包含对应标签的图像文件
   - 根据标签文件中的"ng"字段进行OK/NG分类
   - 按指定比例划分训练集、验证集和测试集
   - 自动复制文件到对应的数据目录结构

2. **检测功能**：
   - 支持多种图像格式(.bmp, .jpg, .jpeg, .png, .gif)
   - 实时显示检测结果及置信度
   - 可视化展示OK/NG概率分布
   - 记录操作日志与检测结果
